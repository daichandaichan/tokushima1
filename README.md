# 徳島建築営業支援ポータル

## 概要
徳島県内の建築工事に関する最新の入札情報、計画案件、および設計事務所情報を自動で収集・配信する営業支援ツールです。

**特徴:**
- ✅ 平日の8時、12時、16時に自動更新
- ✅ 徳島県・徳島市の最新入札情報をリアルタイム配信
- ✅ 建築工事一式に特化した案件抽出
- ✅ 主要設計事務所の詳細情報と営業戦略を掲載
- ✅ 各工法の公式サイトへのワンクリックリンク
- ✅ 完全無料・永久運用

## クイックスタート

### 1. このリポジトリをフォーク
GitHub上でこのリポジトリをフォークし、自分のアカウントにコピーします。

### 2. GitHub Pagesを有効化
1. リポジトリの **Settings** タブをクリック
2. 左メニューから **Pages** を選択
3. **Branch** を `main` に設定して **Save** をクリック
4. 数分後、`https://[あなたのユーザー名].github.io/[リポジトリ名]/` でサイトが公開されます

### 3. 自動更新を有効化（重要）
1. **Settings** ＞ **Actions** ＞ **General** を開く
2. **Workflow permissions** を **「Read and write permissions」** に変更
3. **Save** をクリック

これで完了です！毎日自動でサイトが更新されます。


## 更新スケジュールの変更方法
`.github/workflows/update-tenders.yml` の `cron` 値を編集することで、更新頻度を変更できます。

**例:**
- 毎日午前9時: `cron: '0 0 * * *'`
- 平日の8時、12時、16時: `cron: '0 23,3,7 * * 1-5'` （UTC基準）
- 1時間ごと: `cron: '0 * * * *'`

## トラブルシューティング

### サイトが更新されない場合
1. **Workflow permissions** が「Read and write」に設定されているか確認
2. GitHub の **Actions** タブで、ワークフローが正常に実行されているか確認
3. 実行ログにエラーが表示されていないか確認

### 案件情報が表示されない場合
- 徳島県・市の入札情報サービスが一時的に利用不可の可能性があります
- 翌日の自動更新を待つか、手動で各自治体の入札ページを確認してください

## 営業活動への活用方法

### 1. 最新案件の確認
毎日の朝礼時に、「自動取得案件」タブから最新の入札情報を確認します。

### 2. 設計事務所への営業
「主要設計事務所リスト」から、該当する事務所を特定し、営業戦略に基づいてアプローチします。

### 3. 工法の提案
各案件に対して、「主要営業工法」セクションのリンクから工法の詳細資料を取得し、営業資料として活用します。

### 4. 自治体リンクの活用
「自治体別・建築工事リンク集」から、各自治体の入札ページへ直接アクセスし、詳細な公示内容を確認します。
